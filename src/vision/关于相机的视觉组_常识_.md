<h1 id="jHjkq">1.快门</h1>
**指的是控制光线进入相机感光元件时间长短的机制**。

| 快门速度 | 感光时间 | 亮度效果 | 动态效果 |
| --- | --- | --- | --- |
| **快（短）** | 时间短 | 图像更暗 | 动态冻结、不模糊 |
| **慢（长）** | 时间长 | 图像更亮 | 运动物体会拖影、模糊 |


<h1 id="CLItK">2.曝光</h1>
**指相机感光元件接收的光线量，**

**曝光 = 光线强度 × 快门。**

<h1 id="Tk0ZV">3.增益</h1>
**指对相机感光信号进行电子放大的过程，用来提高图像亮度，但会带来噪点。**

<h1 id="yXtk6">9.等效焦距</h1>
**等效焦距不是镜头真实焦距，而是为了对比视野而换算出来的“参考焦距”，是将不同画幅相机的视角****<font style="color:#DF2A3F;">统一到 全画幅（35mm相机）标准下的焦距值</font>**，**用于比较不同相机的拍摄视角**。

等效焦距 = 实际焦距 × 裁切系数。

<h1 id="CQo1k">10.相机视场角</h1>
**<font style="color:#F38F39;">指 相机能够看到的场景范围所对应的角度，视场角越大，相机看到的范围越广；视场角越小，画面越窄、看得更远。</font>**

<h1 id="tnNHx">12.畸变矩阵</h1>
(k1,k2,p1,p2,k3)

**<font style="color:#F38F39;">相机的畸变矩阵(Distortion Coefficients）是用于描述 镜头畸变效应的数学模型参数，用于在图像校正中去除镜头产生的几何畸变。</font>**

**为什么相机会产生畸变?**

**<font style="color:#5C8D07;">真实镜头不可能做到完美的几何成像。  
</font>****<font style="color:#5C8D07;">光学成像系统存在结构和制造上的不完美，使光线无法完全按照理想的针孔模型进行投影，因此在图像中出现弯曲、拉伸、变形等现象，这就是 镜头畸变。</font>**

<h1 id="VIuWw">13.深度相机</h1>
**是一种不仅能拍摄普通的二维颜色图像（RGB），还能获取每个像素距离相机的深度信息**（即物体到相机的距离）的相机。



分为三种：



**<font style="color:#07787E;">结构光深度相机：</font>**

**通过 投射仪发射特定的红外点阵或条纹光型，照射到物体表面后发生变形，相机采集这些变形，通过形变量计算物距。**

| **优点** | **说明** |
| --- | --- |
| **深度精度高** | **近距离几毫米级精度** |
| **分辨率较高** | **适合人脸识别、手势追踪** |
| **不依赖纹理** | **黑色墙壁、平面仍可测量** |


| **缺点** | **说明** |
| --- | --- |
| **室外阳光表现差** | **因为红外光易被太阳掩盖** |
| **距离有限** | **通常 0.2m ~ 3m** |
| **对运动物体易失效** | **需要采集完整光型** |


****

**<font style="color:#07787E;">双目立体深度相机：</font>**

**使用 两个相同的摄像头，像人眼，通过同一物体在左右视图的位置差（视差）来计算深度。**

| **优点** | **说明** |
| --- | --- |
| **最像人眼，原理简单** | **基于三角测量** |
| **可远距离测量** | **数米到数十米** |
| **室外能力强** | **不依赖主动光源** |
| **可生成稠密点云** | **适合SLAM与机器人** |


| **缺点** | **说明** |
| --- | --- |
| **平面/无纹理物体难测** | **视差无法计算（黑墙、纯白桌面）** |
| **算法复杂、计算量大** | **消耗CPU/GPU资源** |
| **精度依赖基线和标定** | **标定不准误差大** |


****

**<font style="color:#07787E;">TOF深度相机：</font>**

**向目标发射调制光（一般为红外），测量光往返时间差，计算距离。**

| **优点** | **说明** |
| --- | --- |


| **速度快、延迟低** | **适合实时动作捕捉** |
| --- | --- |


| **抗光干扰能力较强** | **比结构光强** |
| --- | --- |


| **能测运动物体** | **单帧实时深度** |
| --- | --- |


| **简单直接、芯片化强** | **手机自动对焦、机器人避障** |
| --- | --- |


| **缺点** | **说明** |
| --- | --- |
| **远距离精度下降** | **噪声随距离显著增加** |
| **分辨率较低** | **常见 240×180、640×480** |
| **多设备互扰** | **多台TOF会干扰** |
