<h1 id="moD0u">前言</h1>
有的安装完ubuntu后会出现没有蓝牙的情况，原因是网卡型号新，ubuntu22.04没有做适配

这个问题没有完全解决，下面是一些尝试的方案



<h2 id="dGxez">查看网卡型号</h2>
```bash
sudo lspci | grep -i network
```

一般为第一个 ，我的是ax210

现在没有蓝牙的一般为mt7925e(他会显示7925什么的），还有Realtek（小螃蟹）网卡 rtl 8992

![](https://cdn.nlark.com/yuque/0/2025/png/62863898/1763862828442-0c1c54cd-481c-4b8c-b424-54a1e2654ada.png)

<h2 id="CNlOy">方案一 运行以下代码</h2>
这个方案有一些用，但不多，有这问题的可以尝试一下

```shell
sudo rmmod btusb
sleep 1
sudo modprobe btusb
sudo service bluetooth start
rfkill block bluetooth
rfkill unblock bluetooth
```

<h2 id="TRDs6">方案二 升级Linux内核</h2>
我看到一些资料,他说要到6.13上才支持mt7925e的全部蓝牙功能，但我更新内核后，对mt7925e不管用，所以对于mt7925e不用更新内核了

一般小螃蟹网卡是升级内核就可以支持蓝牙了





```bash
sudo add-apt-repository ppa:cappelikan/ppa -y
sudo apt update
sudo apt install mainline -y

```

选中一个安装即可

![](https://cdn.nlark.com/yuque/0/2025/png/62863898/1763863152159-fc00f17d-655b-4444-8fcc-20ec2bc1f1dd.png)



<h3 id="pXilT">删除内核  </h3>
<font style="background-color:#FBDE28;">不要删掉正在运行的内核</font>

```bash
sudo dpkg --list | grep linux-image
```
