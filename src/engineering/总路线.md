<h3 id="yzDlg">工程组学习顺序（实时更新）</h3>
>   梳理从基础到机器人开发的整体学习路线，确保各阶段知识连贯、可实操、可落地
>

<h4 id="UsXU9">一、C++</h4>
>   企业级开发基础，后续 ROS2、MoveIt2 必须依赖
>

**学习重点：**

+ 语法基础、指针和引用
+ 面向对象
+ STL 容器与迭代器
+ 异常处理、智能指针、内存管理
+ 多线程与线程池、锁机制
+ C++17/20 常用特性

**建议：**  
 把语言基础打牢，再学习工程化写法（如 RAII、封装、典型模块设计方式）

<h4 id="kcHAB">二、CMake</h4>
>   ROS2 使用 CMake 作为构建系统，必须熟悉才能写包、写库、管理依赖
>

**学习重点：**

+ CMakeLists 基本结构
+ target_xxx 系列命令（target_link_libraries、target_include_directories）
+ add_library、add_executable 的典型用法
+ 安装与导出（install/export）
+ 如何编写第三方库依赖

**建议：**  
 结合简单 C++ 项目实操，并能看懂 ROS2 包的 CMakeLists

<h4 id="dzjkd">三、熟悉 Linux</h4>
>   ROS2 = Linux 环境下的工程框架，操作系统基础越稳越高效
>

**学习重点：**

+ 文件系统、权限、可执行文件
+ bash 基础（cd、ls、grep、find、htop、mkdir、touch）
+ 多进程、多线程观察
+ apt/pip/wget/ssh 等工具
+ vscode + clangd 调试环境

**建议：**  
 尽量使用命令行完成所有操作，提高工程效率

<h4 id="v0dAU">四、Git</h4>
>   工程协作、版本管理、多人开发必备技能，ROS2 工程中也大量依赖 Git
>

**学习重点：**

+ Git 基本概念：工作区、暂存区、版本库
+ 常用指令：add、commit、push、pull、clone、status、log
+ 分支管理：branch、checkout、merge、rebase 的使用场景
+ 解决冲突的方法与实践
+ 使用 GitHub/Gitee 进行代码托管
+ .gitignore 的配置方法

**建议：**

+ 从本地仓库练习，再过度到远程仓库
+ 尝试在大型项目中建立 feature 分支、多人协作、解决冲突
+ 在 ROS2 工程中掌握分支隔离（功能包版本管理非常实用）

<h4 id="KSSgC">五、ROS</h4>
>   全栈机器人框架核心：通信、节点、话题、服务、动作、参数、launch、rqt、tf2
>

**学习资源：**

+ 赵虚左教程：  
 [[https://search.bilibili.com/all?keyword=ROS%20%E8%B5%B5%E8%99%9A%E5%B7%A6]](https://search.bilibili.com/all?keyword=ROS%20%E8%B5%B5%E8%99%9A%E5%B7%A6])([https://search.bilibili.com/all?keyword=ROS](https://search.bilibili.com/all?keyword=ROS) 赵虚左)
+ ROS2 工程依赖与第三方库说明：  
 [https://fishros.org.cn/forum/topic/2119](https://fishros.org.cn/forum/topic/2119)

**学习重点：**

+ rclcpp 基础、Publisher/Subscriber
+ Service、Action
+ TF2 变换、坐标系树
+ launch 文件（XML 与 Python）
+ 组件化（component）、生命周期节点
+ 与硬件通信（串口、USB、TCP）

**建议：**  
 从入门包开始，做到能独立写一个完整功能包（订阅、发布、参数、定时器）

<h4 id="z3MiG">六、MoveIt2</h4>
>   ROS2 的运动规划框架，实现路径规划、逆解、控制执行、碰撞检测
>

**资源：**

+ 入门视频：  
 [https://www.bilibili.com/video/BV1rdxdzkES5?p=14](https://www.bilibili.com/video/BV1rdxdzkES5?p=14)  
 其中 launch 为 XML，可自行改写为 Python
+ 文档与教程
+ 资料：  
 [https://www.cnblogs.com/ai-ldj/p/18370327](https://www.cnblogs.com/ai-ldj/p/18370327)  
 [https://zhuanlan.zhihu.com/p/700147782](https://zhuanlan.zhihu.com/p/700147782)

**学习重点：**

+ MoveGroupInterface
+ Planning Pipeline
+ 规划场景（PlanningScene）与碰撞对象
+ 逆运动学 IK（IKFast）
+ RViz2 可视化
+ 自定义抓取、放置、执行流水线

**建议：**  
 能写一个：

+ 机器人启动 launch
+ Moveit2 配置包
+ 规划与执行 demo

<h4 id="NCm04">七、ros2_control</h4>
>   机器人底层控制框架：关节接口、控制器管理、硬件插件
>

**资源：**

+ 教程视频：  
 [https://www.bilibili.com/video/BV1ku411G7UR](https://www.bilibili.com/video/BV1ku411G7UR)

**学习重点：**

+ hardware_interface::SystemInterface
+ 控制器（joint_state_broadcaster、joint_trajectory_controller）
+ 控制器切换与参数
+ 与 MoveIt2 的执行链路

**建议：**  
 做一个能驱动仿真机器人关节的 ros2_control 插件，再扩展到真实硬件

<h4 id="UoVzZ">八、机器人学</h4>

>

**学习方向：**

+ 机器人学导论
+ 正运动学、逆运动学
+ 雅可比矩阵
+ 奇异点
